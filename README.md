# Docker bazel image with 32 bit executable support

## Build

```bash
export PROJECT_ID=example-gcr-project-id
./build.sh
```

## Run

```bash
export PROJECT_ID=example-gcr-project-id
./build.sh && ./run.sh
```

## Release

```bash
export PROJECT_ID=example-gcr-project-id
./build.sh && ./release.sh
```
